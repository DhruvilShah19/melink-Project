Errors Encountered

CORS Issues: I faced cross-origin resource sharing (CORS) issues while trying to make API requests to external servers. I learned how to mitigate this problem using browser extensions like "Allow CORS."

Real-time Character Matching: Implementing real-time character matching in the Typing Challenge was challenging. I learned how to track user input, compare it with the displayed text, and update the text's color accordingly.

Error trying to diff '[object Object]: Only arrays and iterables are allowed," typically occurs when you're trying to iterate over an object using the *ngFor directive in your Angular template, but Angular expects an array or iterable to be provided.

Binding and Version issues: Can't bind to 'ngValue', 'ngModel' since it isn't a known property of 'option'.ngtsc(-998002)
